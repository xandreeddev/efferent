The same shape as the [dice agent](/efferent/examples/dice-agent/), with **two** tools (`add`,
`multiply`) in one `Toolkit`. The only differences: pass more tools positionally to `Toolkit.make`,
and bind one handler per tool name. The config, loop, and composition root are unchanged.

```sh
bun examples/calcAgent.ts
```
