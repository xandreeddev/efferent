against a logged-in provider — no coding toolkit, no scope runtime, no sub-agents. The
[Your first agent](/docs/your-first-agent/) guide walks through it piece by piece; below is the
**complete, runnable file** exactly as it ships in the repo.

```sh
bun examples/diceAgent.ts
```
