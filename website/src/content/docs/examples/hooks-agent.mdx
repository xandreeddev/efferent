`AgentHooks` let the application watch and influence the loop without owning it. This example logs each
turn and tool result (observe) and **blocks** a tool call that violates a guardrail (steer) —
`onBeforeToolCall` returns `{ action: "block", reason }`, and the reason is handed back to the model as
a tool result so it adjusts in the same turn.

```sh
bun examples/hooksAgent.ts
```
